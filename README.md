# Undivided Clothing Ecommerce Website

A full stack ecommerce website built for a clothing business.

## Tech Stack
- React (Frontend)
- Flask (Backend)
- SQLite (Database)
- Axios (API communication)

## Features
- Product listing from database
- Add to cart
- Remove items from cart
- Clear cart
- Place order
- Orders saved in database

## How To Run Project

### Backend
1. Go to backend folder
2. Activate virtual environment
3. Run:
python app.py

### Frontend
1. Go to frontend folder
2. Run:
npm start
